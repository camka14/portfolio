# Portfolio — Camka

Personal portfolio built with Next.js to showcase projects, including Razumly MVP—the multi-sport event platform for organizing pickup games, leagues, and tournaments. The site highlights live links, previews, and project write-ups.

## Projects highlighted
- **Razumly MVP (multi-sport)**: Scheduling, team management, payments, and chat powered by Appwrite + Next.js. Live at https://mvp.razumly.com with OG previews generated at `/opengraph-image`.
- Additional projects and case studies presented as cards with external links and media previews.

## Tech stack
- Next.js 14 (App Router) with TypeScript
- Tailwind/Mantine (depending on the section) for fast UI layout and theming
- Deployed on Vercel (static-friendly, server components where needed)
